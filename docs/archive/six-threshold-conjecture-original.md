## Section: "The Six-Threshold Conjecture"

Systems capable of self-reference must cross six critical thresholds.

| # | Name | Description |
|---|------|-------------|
| 01 | Differentiation | Emergence of distinct observables |
| 02 | Integration | Formation of coherent structures |
| 03 | Recursion | Self-referential loops appear |
| 04 | Stabilization | Fixed points in dynamics |
| 05 | Reflection | System observes its own state |
| 06 | Encoding | Self-representation achieved |
